## Turquoise template

Адаптивная вёрстка макета лендинга из psd шаблона.

### При разработке пользовался:
* CSS фреймворк SASS
* Нейминг BEM
* HTML шаблонизатор Panini
* JQuery
* Bootstrap 4
* Task manager Gulp 4
### Fonts:
* Open Sans
* Roboto
